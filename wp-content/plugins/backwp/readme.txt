Backup your wordpress site files and databases.

== Description ==

**Main Features:**

1. Select to backup database or files.
2. Monitoring complete backup process. 
3. List backup archive files detailed infomation, like created date, file size, backup type.

**More info:**

* Find out more custom features on [BackWP](http://wphobby.com/)

== Installation ==

1. Upload the contents of backwp.zip to your plugins directory.
2. Activate the plugin
3. There will be a "Backwp" menu on your admin panel. 
4. That's all. You can backup your wordpress files and database now!
